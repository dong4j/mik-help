🖼️ Markdown Image Kit

在 JetBrains IDE 中高效管理 Markdown 图片 | 2.0.0 全新版本

专为 JetBrains 全系列 IDE 设计的 Markdown 图片管理插件，让你在 IDE 中优雅地处理 Markdown 文档中的图片。支持一键上传、剪贴板监听、图床迁移、批量处理等强大功能。

✨ 一键上传 📦 批量处理 🔄 图床迁移 📋 剪贴板监听 🖼️ 图片压缩 ⚡ 异步处理 🔒 安全可靠
⚡

一键上传替换

自动检测文档中的本地图片，一键上传到图床并替换为 markdown 图片标记。支持批量处理，极速高效。

📋

剪贴板监听

复制图片后直接粘贴，自动上传并插入图片链接。无需打开浏览器，专注写作不中断。

🖼️

图片压缩

上传前自动压缩图片，减少存储空间和流量消耗。保持高质量的同时优化文件大小。

🌐

多图床支持

支持自定义图床、PicList、PicGo 等。灵活配置，轻松切换，再也不用担心图床限制。

🔄

图床迁移

一键将图片从旧图床迁移到新图床。数据不丢失，路径不失效，迁移零风险。

🎯

智能处理

单独处理某个图片，批量替换标签，随心所欲。不阻塞主线程，IDE 运行流畅如丝。

📚 快速开始

1

安装插件

在 IDE 的插件市场搜索 "Markdown Image Kit"，或者直接下载 zip 包离线安装

2

配置图床

在设置中配置你喜欢的图床（推荐使用 PicList），或者配置直连 OSS

3

开始使用

复制图片后直接粘贴，或者右键选择上传选项，即可自动上传并插入 Markdown 链接

4

高级功能

使用批量上传、图床迁移等高级功能，提升你的 Markdown 写作效率
