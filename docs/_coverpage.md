# Markdown Image Kit

> 专为 JetBrains IDE 设计的 Markdown 图片管理插件

- 🚀 图片上传：支持粘贴上传、单个上传、批量上传
- 🔄 图床迁移：轻松将图片从一个图床迁移到另一个
- 🎨 图片增强：支持压缩、重命名、水印、格式转换
- 🔗 标签转换：自动转换图片标签格式
- 📦 多图床支持：支持主流图床服务
